# High-Resolution Logo for Logs

This directory contains high-resolution PNG versions with transparent backgrounds for use in logs, documentation, and high-quality printing:

- `logo-highres-2048.png` - High resolution logo at 2048px width
- `logo-highres-4096.png` - Ultra high resolution logo at 4096px width
- `logo-square-highres.png` - Square icon version at 2048x2048px

All files have transparent backgrounds and are suitable for printing and high-quality display.
