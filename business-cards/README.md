# Business Card Logo Versions

This directory contains logo versions sized for standard business cards (3.5" x 2" or 88.9mm x 50.8mm):

- `business-card-front.svg` - Front design in SVG format (scalable)
- `business-card-back.svg` - Back design in SVG format (scalable)
- `business-card-front-print.png` - Front design for printing (1050x600px at 300 DPI)
- `business-card-back-print.png` - Back design for printing (1050x600px at 300 DPI)
- `business-card-front-web.png` - Front design for web preview (350x200px)
- `business-card-back-web.png` - Back design for web preview (350x200px)

Standard US business card size: 3.5" x 2" (88.9mm x 50.8mm)
At 300 DPI: 1050px x 600px
