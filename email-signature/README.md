# Email Signature Logo

This directory contains compact and lightweight logo versions for email signatures:

- `email-signature.svg` - SVG version (scalable)
- `email-signature-small.png` - Compact version at 300x75px (lightweight, ~10KB)
- `email-signature-medium.png` - Medium version at 400x100px (lightweight, ~15KB)

These versions are optimized to be small in file size while maintaining quality, perfect for email signatures that need to load quickly.
