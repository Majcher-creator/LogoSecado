# Banner Logo Versions

This directory contains wide format banners suitable for web or printed use:

- `banner-horizontal.svg` - Horizontal banner in SVG format (scalable)
- `banner-horizontal-web.png` - Horizontal banner optimized for web (1200x300px)
- `banner-horizontal-print.png` - Horizontal banner for print (3600x900px at 300 DPI)
- `banner-vertical.svg` - Vertical banner in SVG format (scalable)
- `banner-vertical-web.png` - Vertical banner optimized for web (600x800px)
- `banner-vertical-print.png` - Vertical banner for print (1800x2400px at 300 DPI)

Use SVG versions when possible for best quality. PNG versions are provided for compatibility.
