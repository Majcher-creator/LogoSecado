# LogoSecado

## Logo Versions for Different Use Cases

This repository contains various versions of the SeCaDO logo optimized for different use cases and platforms. All versions maintain the original design while being adapted for specific resolutions and dimensions.

### Directory Structure

```
├── website/              - Web-optimized versions (favicons, headers, icons)
├── logs/                 - High-resolution PNGs with transparent backgrounds
├── banners/              - Wide format banners for web and print
├── email-signature/      - Compact and lightweight versions for email
└── business-cards/       - Standard business card dimensions (3.5" x 2")
```

### Usage Guidelines

#### Website Version (`website/`)
- **Favicons**: Use `favicon-16x16.png`, `favicon-32x32.png`, or `favicon-64x64.png` for browser favicons
- **Header**: Use `logo-header.svg` or `logo-header.png` for website headers
- **Social Media**: Use `logo-icon.png` (512x512px) for social media profiles

#### High-Resolution Logs (`logs/`)
- **Documentation**: Use `logo-highres-2048.png` or `logo-highres-4096.png` for high-quality documentation
- **Printing**: Ultra-high resolution versions suitable for large format printing
- **Square Icon**: Use `logo-square-highres.png` (2048x2048px) for high-res square applications

#### Banners (`banners/`)
- **Web Banners**: Use `banner-horizontal-web.png` or `banner-vertical-web.png` for web use
- **Print Banners**: Use `banner-horizontal-print.png` or `banner-vertical-print.png` for high-quality printing (300 DPI)
- **Scalable**: SVG versions available for custom sizes

#### Email Signature (`email-signature/`)
- **Small**: Use `email-signature-small.png` (300x75px) for compact email signatures
- **Medium**: Use `email-signature-medium.png` (400x100px) for larger email signatures
- Both versions are optimized for small file size (~10-15KB) to load quickly in emails

#### Business Cards (`business-cards/`)
- **Standard Size**: All versions sized for US standard business cards (3.5" x 2" / 88.9mm x 50.8mm)
- **Print**: Use `business-card-front-print.png` and `business-card-back-print.png` (1050x600px at 300 DPI)
- **Web Preview**: Use `business-card-front-web.png` and `business-card-back-web.png` for web previews
- **Scalable**: SVG versions available for professional printing

### Original Design

The original logo design features:
- **Colors**: Burgundy (#8B1F35), Yellow (#FFC627), and Green (#9ACD32)
- **Theme**: Roofing company with house and sun imagery
- **Company**: SeCaDO - Śląskie Centrum Dachowe (Silesian Roofing Center)

All versions maintain these design elements while being optimized for their specific use cases.

### File Formats

- **SVG**: Scalable Vector Graphics - use when possible for best quality at any size
- **PNG**: Portable Network Graphics with transparency - use for web and digital applications
- **High-DPI**: Print versions are created at 300 DPI for professional quality
