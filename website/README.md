# Website Logo Versions

This directory contains logo versions optimized for web display:

- `logo-header.svg` - SVG version for website header (scalable, perfect quality)
- `logo-header.png` - PNG version at 1200x300px for website header
- `favicon-16x16.png` - Favicon at 16x16px
- `favicon-32x32.png` - Favicon at 32x32px
- `favicon-64x64.png` - Favicon at 64x64px
- `logo-icon.png` - Square icon version at 512x512px for social media

All PNG files are optimized for web use with transparency.
